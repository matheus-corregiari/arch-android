# BaseRecyclerAdapter

abstract class [BaseRecyclerAdapter](index.md)&lt;[MODEL](index.md) : [Any](https://kotlinlang.org/api/core/kotlin-stdlib/kotlin/-any/index.html)&gt;(differ: [DiffUtil.ItemCallback](https://developer.android.com/reference/kotlin/androidx/recyclerview/widget/DiffUtil.ItemCallback.html)&lt;[MODEL](index.md)&gt; = DefaultItemDiffer()) : [RecyclerView.Adapter](https://developer.android.com/reference/kotlin/androidx/recyclerview/widget/RecyclerView.Adapter.html)&lt;[BaseViewHolder](../-base-view-holder/index.md)&gt; , [StickyHeaders](../../br.com.arch.toolkit.android.recyclerAdapter.stickyheader/-sticky-headers/index.md)

A base implementation of [RecyclerView.Adapter](https://developer.android.com/reference/kotlin/androidx/recyclerview/widget/RecyclerView.Adapter.html) that uses [AsyncListDiffer](https://developer.android.com/reference/kotlin/androidx/recyclerview/widget/AsyncListDiffer.html) and [ViewBinder](../-view-binder/index.md) for efficient list updates and simplified view binding.

#### Parameters

android

| | |
|---|---|
| MODEL | The type of data to be displayed in the list. |
| differ | The [DiffUtil.ItemCallback](https://developer.android.com/reference/kotlin/androidx/recyclerview/widget/DiffUtil.ItemCallback.html) to use for calculating list differences. Defaults to [DefaultItemDiffer](../-default-item-differ/index.md). |
