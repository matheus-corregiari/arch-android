# FoldableActivity

[android]\
abstract class [FoldableActivity](index.md) : [AppCompatActivity](https://developer.android.com/reference/kotlin/androidx/appcompat/app/AppCompatActivity.html)

An abstract base [AppCompatActivity](https://developer.android.com/reference/kotlin/androidx/appcompat/app/AppCompatActivity.html) that simplifies handling foldable devices.

It automatically manages layout inflation for different folding states based on the provided orientation and layout IDs.
