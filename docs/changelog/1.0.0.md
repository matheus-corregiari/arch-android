# Changelog - 1.0.0

**Release Date:** 2025-02-20

## Motivation

Publish the first stable Arch Android Toolkit release.

## Notable changes

- added state machines for view and scene transitions
- added typed memory, SharedPreferences, and encrypted storage
- added property delegates for storage, extras, views, and ViewModels
- added RecyclerView adapter and sticky-header utilities
- added foldable layout helpers and application context access
- configured automated build, lint, coverage, documentation, and publishing

## Compatibility notes

- This is the initial stable release.
