This is the initial stable release of the **Arch Android Toolkit**, providing a robust set of utilities for modern Android development with a focus on KMP compatibility and developer ergonomics.

### ‚ú® Features
- **State Machines**: Flexible orchestration for View and Scene transitions (`StateMachine`, `ViewStateMachine`, `SceneStateMachine`).
- **Storage Abstractions**: Typed key-value storage with in-memory and `SharedPreferences` (including Encrypted) implementations.
- **Property Delegates**:
    - **Persistence**: Easy integration of cached/config values with storage.
    - **Android Core**: Delegates for `Bundle` extras, `ViewModel` providers, and `View` finding.
- **Recycler Adapter Utilities**: Simplified `BaseRecyclerAdapter` and `SimpleAdapter` with built-in `ViewBinder` support and sticky headers.
- **Foldable Support**: Helpers and `FoldableActivity` to handle hinge and posture changes using Jetpack WindowManager.
- **Context Provider**: Global `ContextProvider` utility for safe application context access.
