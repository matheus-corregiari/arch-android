# Arch Android Toolkit

Arch Android provides focused Android utilities for projects that use Kotlin, including KMP
projects that consume the library from `androidMain`.

## Modules

The project publishes one module:

```kotlin
implementation("io.github.matheus-corregiari:arch-android:<version>")
```

Use the guides in this site for setup and common workflows. The API reference is generated from
the source KDoc.
